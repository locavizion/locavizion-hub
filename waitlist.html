Deployment Process

How to Update the Live Site
Step 1: Download the updated HTML file from Claude
Step 2: Rename it to index.html
Step 3: Go to your GitHub repository → click on index.html → click the pencil (edit) icon
Step 4: Select all → paste the new code → scroll down → click "Commit changes"
Step 5: Wait 60-90 seconds. GitHub Pages rebuilds automatically. Clear your browser cache and reload locavizion.com to see changes.

No terminal, no Git commands, no deployment tools needed. The entire workflow happens in the GitHub web interface.
